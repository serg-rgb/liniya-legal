Политика конфиденциальности

Приложение: Линия — Футбольная статистика
Редакция от: 17 апреля 2026 года

1. Общие положения

Настоящая Политика конфиденциальности (далее — «Политика») определяет порядок обработки и защиты персональных данных пользователей мобильного приложения «Линия» (далее — «Приложение»).

Оператор обработки данных: физическое лицо, разработчик Приложения (далее — «Оператор»). Контакт для связи: hupernova@gmail.com.

Установка и использование Приложения означает согласие пользователя с условиями настоящей Политики.

2. Какие данные собирает Приложение

2.1. Данные, которые Приложение НЕ собирает

  • Имя, фамилия, отчество
  • Номер телефона
  • Email
  • Адрес, геолокация
  • Платёжные реквизиты
  • Биометрические данные
  • Данные о контактах, SMS, звонках, фото, видео

2.2. Данные, которые сохраняются локально на устройстве пользователя

  • Список избранных футбольных команд (только ID и название)
  • Настройки темы оформления (светлая/тёмная)
  • Флаг прохождения приветственного экрана

Эти данные хранятся в локальном хранилище устройства (Android DataStore), не передаются Оператору и удаляются при удалении Приложения.

2.3. Данные, передаваемые на серверы Оператора

При использовании Приложения происходят анонимные запросы к API-серверу Оператора для получения:

  • Расписания матчей
  • Результатов и статистики
  • Коэффициентов букмекеров (из открытых источников)
  • AI-прогнозов (запрашиваются только по инициативе пользователя)

С этими запросами не передаётся никакая личная информация. На сервере фиксируется только IP-адрес пользователя в стандартных логах веб-сервера сроком до 30 дней для целей технического обслуживания и защиты от атак.

3. Цели обработки данных

  • Предоставление пользователю актуальной футбольной статистики
  • Функционирование избранных команд и уведомлений о голах
  • Обеспечение технической работоспособности Приложения
  • Защита от злоупотреблений и мошенничества

4. Третьи стороны

Приложение получает данные от следующих внешних источников:

  • API-Football (api-sports.io) — данные о матчах, командах, игроках
  • The Odds API (the-odds-api.com) — коэффициенты букмекеров
  • OpenRouter (openrouter.ai) — инфраструктура для AI-прогнозов

Пользовательские данные в эти сервисы не передаются. Передаются только обезличенные параметры запроса (ID лиги, дата матча и т.п.).

5. Реклама

Приложение может содержать рекламные блоки, предоставляемые рекламными сетями (например, Яндекс Реклама). Рекламные сети могут использовать обезличенный рекламный идентификатор устройства (GAID) для показа релевантных объявлений. Пользователь может сбросить рекламный идентификатор в настройках операционной системы Android.

6. Push-уведомления

Приложение может отправлять локальные push-уведомления о голах избранных команд. Эти уведомления формируются локально на устройстве пользователя на основе данных с сервера Оператора. Push-серверы (Firebase, RuStore Push) не используются.

Пользователь может в любой момент отключить уведомления в настройках устройства.

7. Безопасность

Передача данных осуществляется по протоколу HTTP (техническая особенность MVP-версии). В будущих версиях планируется переход на HTTPS. На сервере Оператора применяются стандартные меры защиты: брандмауэр, ограничение доступа, регулярное обновление ПО.

8. Права пользователя

В соответствии с Федеральным законом №152-ФЗ «О персональных данных» пользователь имеет право:

  • Получать информацию о своих данных у Оператора
  • Требовать удаления своих данных
  • Отзывать согласие на обработку

Для реализации этих прав направьте запрос на email: hupernova@gmail.com.

9. Возрастное ограничение

Приложение предназначено для пользователей старше 18 лет в связи с отображением коэффициентов букмекеров. Приложение носит исключительно информационный характер и не принимает ставки, не является букмекерской конторой и не стимулирует участие в азартных играх.

10. Изменения Политики

Оператор оставляет за собой право изменять настоящую Политику. Новая редакция вступает в силу с момента публикации в Приложении и по ссылке в карточке приложения в RuStore.
